# Weather API Project

A simple Python command-line app that fetches current weather data using the Open-Meteo API.

## Features

- Fetch current temperature
- Fetch wind speed
- Display clean terminal output
- Uses a free public API with no API key

## Tech Stack

- Python
- Requests
- Open-Meteo API

## How to Run

```bash
pip install -r requirements.txt
python main.py
